## 環境構築
1. .env.exampleファイルをコピーして.envファイルを作成
2. .envの定義を変更
3. 依存関係のパッケージをインストール
    ```sh
    pnpm install
    ```
4. 開発サーバーを起動
    ```sh
    pnpm run dev
    ```
